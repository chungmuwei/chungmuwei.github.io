Tracing Code

Desk check: write out the state(value) of variables as you go though loop and if/else By doing this, the chance of identifying a problem is very high

Different types of testing

  1. Black box testing: input -> black box -> output Not looking at the source code Writing test cases without any knowledge of the code that will be tested
  1. White box testing we can see the codes
